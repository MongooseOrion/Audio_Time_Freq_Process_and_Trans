# 基于 FPGA 的声音处理系统

本项目用于降低原始音频中的声学回声，并可修改人声音色。采用实时流处理。

## 仓库结构

```
|-- Document                // 相关技术文档和说明
|-- RTL                     // RTL 代码
|-- Software                // Matlab 仿真代码和相关系统控制程序
```

## 系统参数规格和功能指标

| 项目 | 参数 |
| :--: | :---: |
| 输入采样频率 | 48kHz |
| 输出采样频率 | 48kHz |
| 输入采样位深（量化位宽） | 16bit |
| 输出采样位深（量化位宽） | 16bit |
| 输入通道数 | 2-left, right |
| 输出通道数 | 2-left, right |

受支持的功能：

  * 人声实时调整；
  * 抑制音频回音效果；
  * 音频中背景声抑制；
  * 分离背景声和人声；
  * 音频的流式传输（UDP）；
