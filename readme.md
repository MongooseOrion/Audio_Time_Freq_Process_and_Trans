# 基于 FPGA 的声音处理系统

本项目用于降低原始音频中的声学回声，并可修改人声音色。采用实时流处理。

## 仓库结构

```
|-- Document                // 相关技术文档和说明
|-- RTL                     // RTL 代码
|-- Software                // Matlab 仿真代码和相关系统控制程序
```

## 系统输入输出规格

| 项目 | 参数 |
| :--: | :---: |
| 输入采样频率 | 48kHz |
| 输出采样频率 | 48kHz |
| 输入采样位深（量化位宽） | 16bit |
| 输出采样位深（量化位宽） | 16bit |
| 输入通道数 | 2-left, right |
| 输出通道数 | 2-left, right |
